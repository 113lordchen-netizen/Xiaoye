## ✨ Features (核心特性)

### 📱 Fully Responsive (全端响应式架构)
- **One Codebase, All Devices**: 同一套核心代码，智能适配 PC 与 移动端。
- **PC Mode**: 经典的左右分栏布局 + 所见即所得 (WYSIWYG) 编辑 + 拖拽式模板交互。
- **Mobile Mode**: 自动折叠为单栏 App 形态 + iOS 风格悬浮 Dock + 抽屉式菜单。

### 🛠️ Pain Point Solved (痛点狙击)
- **螺旋笔记修复 (Notebook Fix)**: 独家研发的 CSS 逻辑，完美解决市面上“笔记模板”在文字换行时背景线错位的问题。无论输入多少字，虚线永远对齐。
- **Smart Copy (智能复制)**: 一键清洗无关样式，直接粘贴至微信公众号后台，格式不乱。

### 🎨 Premium Aesthetics (黑金美学)
- **Visual System**: 采用 "Black & Gold" 黑金品牌色系，配合马卡龙渐变，提供极度舒适的视觉体验。
- **Detail Oriented**: 从圆角到阴影，每一个像素都经过精心打磨。

## 🚀 Tech Stack (技术栈)

本项目坚持 **"Vanilla Philosophy" (原生主义)**，拒绝臃肿的框架：

- **Core**: HTML5, CSS3 (Flexbox/Grid), Vanilla JavaScript (ES6+)
- **No Dependencies**: 0 依赖，0 框架，秒级加载。
- **Privacy**: 所有数据在浏览器本地内存处理，无服务器上传，绝对隐私安全。

## 📖 How to Use (使用指南)

1. **选择模板**: 点击左侧 (PC) 或左上角菜单 (手机) 选择心仪的模板。
2. **编辑内容**:
   - **PC端**: 直接在右侧预览区点击修改文字。
   - **手机端**: 切换到"代码模式"输入内容，点击"预览"查看效果。
3. **一键复制**: 点击底部的 `📋 复制` 按钮。
4. **粘贴发布**: 前往微信公众号后台 `Ctrl+V`。
